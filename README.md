# pygard-site
🛡️ PyGuard Pro

PyGuard Pro est un antivirus léger et performant développé en Python, conçu pour offrir une protection efficace contre les menaces numériques grâce à une analyse par signatures et une vérification cloud.

✨ Fonctionnalités

Analyse en temps réel : Surveillance des fichiers et des processus.

Base de données de signatures : Détection locale rapide des malwares connus.

Cloud Threat VT : Intégration avec des services cloud pour une détection avancée.

Interface intuitive : Gestion simplifiée des menaces et rapports de scan.

Liste blanche : Possibilité d'exclure des fichiers ou extensions spécifiques.

🚀 Installation

Pour installer PyGuard Pro, téléchargez la dernière version de l'installateur dans la section Releases de ce dépôt.

Téléchargez PyGuard_Pro_Setup.exe.

Lancez l'installateur.

Suivez les instructions à l'écran.

🛠️ Technologies utilisées

Langage : Python

Cryptographie : Hashlib (SHA-256)

Interface : Tkinter / CustomTkinter (selon votre version)

Packaging : Inno Setup / PyInstaller

Note : Ce projet est fourni à des fins éducatives et de sécurité personnelle.
